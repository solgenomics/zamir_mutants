How to search the database


  • Click on the Search the database icon

  • If you know the family ID of a specific mutant line then mark this option and write in the text box the family code
      (each mutant line code begins with an "e" for EMS mutants or an "n" for FN mutants followed by a 4 digit number,
      so if you are interested in family no. 2 from the EMS mutants the family code will be e0002)
  • If you'd like to search according to a keyword, mark this option and type in the keyword that fits the mutant you are interested in.
      (example keywords: dwarf, anantha, ghost, entire)
    You can narrow down your search to a specific phenotypic category by selecting one from the pull down list.
