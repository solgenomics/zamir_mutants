How to search the database


  • Click on Search the database link

    • There are three options for search:

    1. search according to family ID:
      If you know the family ID of a specific mutant line, write in the text box the family code.
      Each mutant line code begins with "e" for EMS mutants or "n" for FN mutants
      followed by a 4 digit number.
      So if you are interested in family no. 2 from the EMS mutants the family code will be e0002

    2. Search according to a keyword:
      type in the keyword that fits the mutant you are interested in.
      (example keywords: wiry, dwarf, anantha, ghost, entire)

    3. Search according to the phenotypic catalog:
      In the phenotypic catalog there are 15 categories
      For each category there are several sub-categories that appear on the adjacent pull-down menu.
      A mutant can belong only to one sub-category in each of the 15 categories

      You must select at least one category. Selecting sub-category is optional.

      When selecting more than one category the search will be for mutants classified to all the
      categories selected (category1 and category2).
      Therefor, when selecting many categories the output will be very few mutants or none.

      It's recommended to search first for one category and then narrowing down the results
      by selecting one sub-category and/or adding another category.



    Search results:

     For each mutant in the search results its "ID card" is printed:

  • Code: a unique code for each mutant in the database ("e0002m1)- begins with "e" for EMS or "n" for fast-neutron mutant, followed by 4-digit number which is the number of the mutant line. The last two characters are "m" + a number which stands for the mutation number within the line. Since each line can segregate for more than one mutation, their codes will be similar except for the last digit.
  • Details: description of the mutant
  • Verified: whether the mutant phenotype was verified in the M3 generation ("yes" or "no")
  • Gene: mode of action of the mutated gene ("recessive, dominant, partial-dominance"). This field does not appear for all mutants.
  • Similar to: A link to the TGRC database. Appears for mutants that show phenotypic similarity to known genes from the TGRC collection
  • Categories: a list all the categories from the phenotypic catalog that the mutant is classified into


    • The description of the mutant is followd by its photo thumbnails (if it has any). Clicking each thumbnail opens a large photo in another window.
